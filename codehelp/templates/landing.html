CodeHelp is a tool that can help you while you code.

It can explain issues and answer questions about your code. It uses GPT-3, a large language model, so it can understand code and natural language queries and respond in natural language as well.

You can:

  • Ask how to start writing a program.
  • Get an explanation of a confusing error message.
  • Ask why your code isn't working correctly in a particular spot.
  • Learn more about specific syntax, functions, or techniques in your programming language of choice.
  • ... and a lot more.
